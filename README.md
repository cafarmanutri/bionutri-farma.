# 🌿 BioNutri Farma

O **BioNutri Farma** é uma solução digital completa para avaliações nutricionais e farmacológicas.

## 🚀 Funcionalidades
- **Cadastro Biométrico:** Nome, idade, peso e altura.
- **Antropometria Completa:** Medição de perímetros (coxa, braço, cintura, etc).
- **Composição Corporal:** Protocolo Pollock de 7 dobras para cálculo preciso de % de Gordura e Massa Magra.
- **Relatórios:** Geração instantânea de PDF clínico para o paciente.

## 🛠️ Tecnologias
- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
- [FPDF](http://pyfpdf.readthedocs.io/)
